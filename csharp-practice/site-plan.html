Overview

Purpose

The purpose of this site is to help beginners learn and practice C# through organized topics, examples, and simple quiz features. It also allows users to track their learning progress and review core C# concepts in a clear, interactive way.

Audience

This site is designed for beginner and intermediate C# learners, including college students, self-taught programmers, and anyone who wants a clean, simple place to review C# basics and track what topics they have mastered.

Dynamic elements

JavaScript will generate C# topic cards dynamically from an array of topic objects. Users will be able to filter topics, mark learning progress (Not Started, In Progress, Mastered), and view progress summaries. A separate quiz page will randomly pull questions from a stored question array, check answers with conditionals, and track the user's score. The project will use ES modules, arrays, objects, event listeners, DOM manipulation, and conditional branching.

Content

Home page

The home page will introduce the site, show a progress summary, provide filter controls, and display all C# topic cards generated with JavaScript.

Practice Page

The practice page will show one quiz question at a time, check the user's answers, give feedback, and track the session score. All questions will be stored in a JavaScript array and pulled randomly.
